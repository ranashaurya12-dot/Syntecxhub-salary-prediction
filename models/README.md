# Salary Prediction Using Machine Learning

## Project Overview
This project predicts employee salary using:

- Experience
- Test Score
- Education Level

## Technologies Used

- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Joblib

## Models Used

1. Single Feature Linear Regression
2. Multiple Linear Regression

## Evaluation Metrices

- RMSE
- R² Score

## Best Model

Multiple Linear Regression was selected as the best model based on R² Score.

## Prediction Example

Experience = 4

Test Score = 52

Education = Bachelors

Predicted Salary =  34000.0
